# Euphoria
An open-source 2D game engine targeted at new game developers trying to learn.
  
# Features
- **2D Physics**: Built on [box2d](https://github.com/erincatto/box2d) which enables realistic 2D physics.
- **Layer System**: Manage all your Games Scripts with an easy to use layer push/pop system.
- **GUI Rendering**: Create Editor UI or Game UI using [ImGui (Docking Branch)](https://github.com/ocornut/imgui/tree/docking)
- **2D Sprite Rendering**: Render any sprite with support for transformations (Position, Rotation and Scale)
- **Input System**: Full Support for Keyboard and Mouse input, and future support coming for Controllers and other Input Devices.

# Libraries
- [ImGui (Docking Branch)](https://github.com/ocornut/imgui/tree/docking)
- [SFML](https://github.com/SFML/SFML)
- [box2d](https://github.com/erincatto/box2d)

# Prerequisites
Before building Euphoria, make sure you have the following installed:
- **Visual Studio 2026 (or 2022)*** (with the C++ Desktop Development module)
- **Git** (for cloning the repository)
